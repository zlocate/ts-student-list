# Пример REST CRUD API (список студентов)

## Описание
Простой CRUD REST API для управления списком студентов.

## Стек
- Nest.js
- MongoDB (Mongoose)
- Typescript 
- Swagger
- ESLint


## Установка

```bash
$ npm install
```

## Запуск сервиса

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Описание API
Автоматически генерируемое Swagger-описание доступно по энтрипоинту [/api](http://localhost:3000/api)


## Тесты

```bash
# unit tests
$ npm run test
```
